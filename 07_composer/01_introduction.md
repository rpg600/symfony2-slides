# What is it ?

* Php dependency manager

# Quick installation

Linux/Mac:

    !bash
    $ curl -sS https://getcomposer.org/installer | php
    $ php composer.phar

Move the files into your bin:

    !bash
    $ mv composer.phar /usr/local/bin/composer
    $ composer
